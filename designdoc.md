 # Приложение chezget

- Приложение написано на языке go.
- Структура исходного кода соответствуюе современным рекомендациям для go.
- Приложение должно полностью собираться инструкцией go build
- Артифакт сборки: исполняемый файл с именем chezget.
- Проект должен собираться и успешно работать под системами: windows, linux, macos, freebsd в архитектурах
  amd64 и arm64
- Исходный текст должен быть покрыт юнит-тестами не менее чем на 60% кодовой базы. Основные и критичные
  участки проекта должны быть покрыты тестами не менее чем на 80%.
- Тесты должны выполняться через вызов go test.
- В корне проекта должен быть файл README.md, содержащий краткое описание назначения проекта и информацию
  о сборке и использовании. Текст файла README.md должен быть написан на английском языке в стиле,
  принятом на платформе github.com
- Исходники проекта распростроняются по лицензии MIT и оформлены в соответствии с требованиями лицензии.
- Разрешается использовать любые внешние зависимости при условии совместимости их лицензии с MIT.
- В корне проекта должен присутствовать файл AGENTS.md с необходимой для агентской разработки информацией.
  при любых изменениях проекта, файл AGENTS.md должен соответствующе актуализироваться.

# Назначение приложения chezget

Приложение chezget является приложением-компаньоном для [chezmoi](https://github.com/twpayne/chezmoi).
Задача chezget заключается в установке приложений, написаных на go и rust, согласно списка, находящегося
под управлением chezmoi.

Список устанавливаемых приложений хранится в конфигурационном файле, согласно спецификации XDG. Формат файла: *.ini

Конфигурационный файл содержит отдельные секции для каждого языка (go, rust).

Каждая запись секции конфигурационного файла содержит ссылку на устанавливаемое приложение в формате,
понятном для менеджера пакетов соответствующего языка (go для go, cargo для rust).

Например:

для записи секции rust 

```
kotlin-lsp
```

Будет выполняться команда

```
cargo install kotlin-lsp
```

а для записи секции go

```
github.com/jesseduffield/lazygit@latest
```

будет выполняться команда

```
go install github.com/jesseduffield/lazygit@latest
```


задача программы chezget заключается в том, чтоб прочитать Конфигурационный
файл и вызвать соответствующие менеджеры пакетов для выполнения ими установки соотвтествующих
пакетов.
